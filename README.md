## EasyEdit

Required build dependacies

- - npm
- - vite
- - file-saver
- - html2canvas
- - jspdf
- - mermaid
- - react
- - react-dom
- - react-markdown
- - remark-gfm
- - electron

<img src="./screenshots/sample001.png" alt="Example" width="500" height="300">

## Run the project

```
$ npm start
```

## Build an standalone App
```
$ npm run electron:build
```
